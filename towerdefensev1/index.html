How to Play

  1. Place towers along the path to stop enemies.
  2. Earn gold by defeating enemies.
  3. Upgrade towers between waves.
  4. Survive all waves to win!

Note: While not mine, the game is very simple, but my goal was to make it a lot more complicated sadly nothing went to plan
